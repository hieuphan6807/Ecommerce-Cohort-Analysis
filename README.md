# GA Sessions 2017 Analysis

This repository contains SQL queries and analysis scripts using the `ga_sessions_2017` dataset on Google BigQuery.  

## Dataset
- **Name:** ga_sessions_2017  
- **Source:** Google BigQuery Public Datasets  
- **Description:** Contains Google Analytics session data for the year 2017, including user sessions, traffic sources, device info, and ecommerce transactions.

## Key Analyses
- Total sessions and users per month
- Traffic source breakdown (organic, paid, referral, etc.)
- Device and browser usage statistics
- Top pages visited and pageviews
- Ecommerce revenue and conversion rates
- Session duration and user engagement metrics
- New vs returning users analysis

## How to Run
1. Connect to BigQuery using your Google Cloud account.
2. Open the `SQL Queries` folder.
3. Run the provided queries on the `ga_sessions_2017` dataset.
